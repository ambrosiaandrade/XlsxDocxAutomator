Tutorial Sobre Configuração

Bem-vindo!

Selecione um fluxo para começar.
O fluxo UPE utiliza modelos institucionais.
O fluxo Genérico permite usar seus próprios modelos.

Para enviar por e-mail, faça a configuração antes de começar.

Fluxo UPE

Suba a sua planilha e gere documentos com base nos modelos institucionais.

Fluxo Genérico

Suba sua planilha e seu próprio modelo de documento (.docx) para personalização.
